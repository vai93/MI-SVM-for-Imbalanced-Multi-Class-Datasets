# MI-SVM: Multiclass Imbalanced Support Vector Machine

## Description
MI-SVM is a multiclass classification approach designed for imbalanced datasets. 
It integrates Support Vector Machines with a kernel transformation technique to enhance minority class representation. 
The method employs pairwise class decomposition, hierarchical classification, and support vector–based boundary detection to improve classification performance.

---

## Features
- Handles multiclass imbalanced datasets
- Kernel Transformation for improved class separation
- Pairwise class decomposition strategy
- Hierarchical classification mechanism
- Support vector–based boundary detection
